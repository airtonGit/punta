A) Folder Structure - top

In the psd folder you will all the PSD files for the template. In the html folder you will all the html files for the template.

B) CSS Files and Structure - top

You should find the main stylesheet of the theme (style.css) in the root folder. All other css files are contained in the folder /css, along with there Sass counterparts. If you have Sass knowledge you should edit your website using Sass, otherwise, it's find to simply edit the css files directly. The HTML templates only load the .CSS files anyway.

In the /colors folder you will find the CSS files that govern the main color of the scheme. If you need to create a new scheme, you can simply open up the default.css file and replace the color values you will find with your own.

C) JavaScript - top

This theme imports works with the following Javascript files.

  1. jQuery
  2. jQuery UI datepicker
  3. jQuery Flexslider
  4. JQuery prettyPhoto (lightbox)
  5. jQuery Superfish
  6. jQuery Equalheights (if you ever need it for equalizing column heighs)
  7. jQuery Dropkick (used for the custom dropdowns)
  8. jQuery html5shiv (for backwards HTML5 elements compatibility)
  9. jQuery fitVids (for responsive videos (add a div.video to your video container)
  10. jQuery mmenu.js (for the mobile menu)
  11. contact.js (for the Ajax contact form)

All javascript files are located in the /js folder

D) Weather Widget - top

Santorini Resort comes with a custom dynamic weather widget that grabs your location and displays its weather along with a nice icon next to it.

To change your location first go to this page: https://www.edg3.co.uk/snippets/weather-location-codes/ and search for your city. You should get a code like GRXX0004 (e.g. the previous code is Athens, Greece.

Then, open the file js/scripts.js and replace the value of line 40 with your own.
